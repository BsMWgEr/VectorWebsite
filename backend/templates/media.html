To Upload a File:

  1. Click the button labeled 'browse' below
  2. Select a file to upload

After following the 2 steps above you will have uploaded the selected file to your Websites main public storage (aka Amazon Cloud). This File is instantly viewable from the link provided after uploading. At the same time, this will automatically create a new database item under the Images section which (left sidebar in the Admin) is attachable to any Inventory Item. You can attach this image by selecting it from the dropdown when building a new Inventory Item (Newly uploaded items automatically propagate labeled as the file name you upload with example --> 'new_image.jpg'. You can find this Image information under the Images section on the left side of your admin page

All files uploaded are viewable to anyone on the internet. DO NOT upload private files or photos via this method! (There will be a private upload file section soon!)

TO ACCESS BUCKET DIRECTLY: Login to Bucket with the first link (S3 Bucket Login), after logged in click the second link (StockVectorRigs S3 Bucket) OR search for s3 in the AWS Console search bar at the top of page after login. Select 'stockvectorigs' one under the bucket tab

NO SPACES IN File Name! (this can be fixed later, but for now. No spaces please!)
